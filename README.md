##  サービス概要

『Petit Marché』は、『お弁当』や『スイーツ』などあらゆるコンビニ商品を画像付きで投稿しあう情報交換アプリです。<br>

## 想定されるユーザー層<br>

コンビニが良いが好きで、新製品はチェックしている人<br>

## サービスコンセプト

市街地でも田舎でも外出時に見かけないことはないと言えるくらいに存在する「コンビニ」。<br>
食料品から日用雑貨まで取り揃えてあるコンビニはどんどん店舗を増やし街中では溢れんばかりとなっています。<br>
そんなコンビニにおいては店舗だけではなく商品の入れ替わりも非常に激しく、各社競い合うかの様に消費者の目を引く商品を次々と生み出しています。<br>
来店客がどのような商品を求めているのか・・・？その時の流行に乗っかった商品、春や夏など季節に合わせた商品、或いは復刻品で懐かしさを出したものや爆盛りなど奇抜さを狙った商品など工夫を凝らした商品が次々と登場し、お客さんに対してマンネリ感を出させない努力も伺えます。<br>
ただ、利用客側からしてみると完全な新商品というのは未知の存在。<br>
新しい物が好きで新商品があると思わず目に止まってしまうが、果たしてこれは当たりなのかハズレなのか・・・そのことが頭をよぎり結局無難な商品へ行ってしまった経験は誰しもあるのではないでしょうか？<br>
新商品のレビューが一括で見られるような物があれば・・・
- 購入するかどうか迷っていたけど、他の人のレビューを見て美味しそうだし買ってみよう！と購入に踏み切れる場面が発生したり、自分が購入した新商品を気に入ったものだったらお勧めしたり、逆にリピート無し💦な商品だったら注意喚起したりするなど利用客の購入の判断材料となる
  
- レビューの得点など数値化することによってどの新商品が人気なのか見える化に繋がり、人気・不人気が分かりやすくなる
  
- 検索機能や履歴を追うことで各社どんな方向性でどんな新商品を取り揃えてきているのか・・・一目でわかるようになってくる。

色々分からない・知らなかったことをどんどん共有できれば、よりコンビニライフが便利になりそうかな・・・という思いから作成に取り掛かりました。
<br>

### 本サービスの機能は以下の通りです。（実装予定も含む）

・全国各地のコンビニから購入した商品をここが良かった！逆にここは改良して欲しいって思えるレビューや写真を投稿できます。<br>
・投稿時はタイトル、商品カテゴリ、商品名、価格、購入日、コンビニブランドや写真も投稿できます。購入先店舗はGoogleMap API を利用し、地図上から店舗を選ぶ仕組みとなっています。<br>
・タイトル、商品カテゴリ・コンビニブランドタグでマルチ検索ができます。<br>
・お気に入りの投稿に対してはいいねを送ることができます。<br>
・レビュー投稿済みユーザーに対して各種ランキングページが閲覧できる様になります。<br>
・ユーザー詳細ページでは今までの投稿が一覧で確認・編集・削除が出来ます<br>
<br>

### 他のサービスと比較して差別化できるポイントは以下の通りです。

類似サービスとして、以下のサービスがあります。<br>
・[Google Mapのクチコミ](https://www.google.com/maps)<br>
・[Xの投稿](https://x.com/)<br>
対して、本サービスの差別化ポイントは以下の通りです。<br>
<br>
・コンビニ特化： コンビニの商品に特化しており、店内で扱っている商品にスポットを当てて提供しています。新製品から定番商品まであらゆる商品をレビューして、コンビニ商品の最新情報を提示できます。<br>
・新製品情報共有： コンビニを訪れた全国の消費者が実際に購入し、思い思いにレビューした情報を共有する情報交換サービスです。<br>
・発売されたばかりの新商品も実際の生の声として情報を提供し、全国の消費者に届けます。<br>
・レビュー内容: いつ・どこで・どのような商品を・どのくらいの値段で購入したか・・・などレビュー内容の他に一つの商品に対して詳細な情報を入力することができます。<br>
・各種ランキング： いいね数が多い順・投稿が多いカテゴリランキング・投稿数が多いユーザランキングなど投稿済みユーザーに対してランキング機能を導入する予定です。<br>
・タグ付けとマルチ検索： レビューにタグを付けマルチ検索を提供し、ユーザーが効果的に情報を見つけやすくします。<br>
・X 共有機能： お気に入りのレビューを他のユーザーと共有できる機能を提供予定です。コンビニ商品に敏感なユーザー同士の交流と情報共有が促進されます。<br>

「Petit Marché」は、コンビニ商品の魅力や新しい情報をユーザーに提供することを目指します。<br>
<br>
